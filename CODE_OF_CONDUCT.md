# Code of Conduct

This applies to all interactions in this repository — Issues, Discussions, and comments.

## Expected

- Be respectful and constructive.
- Stay on topic: Zenvara CLI usage, bugs, and feature requests.
- Assume good faith; disagree with ideas, not people.
- Redact secrets, credentials, and personal data from logs and examples you post.

## Not tolerated

- Harassment, insults, or discriminatory language.
- Trolling, spam, or off-topic promotion.
- Publishing others' private information.

## Enforcement

Violations may result in content removal, and temporary or permanent bans from this repository. Report unacceptable behavior via [zenvara.ai/contact](https://zenvara.ai/contact).
